🔑 GitHub Personal Access Token unset

One-time setup for one-tap commits. Without a PAT, you'll use copy/paste mode.

How to create a PAT (1 min)
  1. Open github.com/settings/tokens (fine-grained)
  2. Click Generate new token
  3. Name: "Worldcup Pool Commish" · Expiration: pick any (longer = less hassle)
  4. Repository access: Only select repositories → Worldcup-pool
  5. Permissions → Repository permissions → Contents: Read and write
  6. Generate token, copy it, paste below.
The token is stored ONLY in your browser's localStorage. Never sent anywhere except api.github.com.
